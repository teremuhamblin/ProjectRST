Introduction
============

ProjetRST est un projet structuré autour d'une documentation entièrement en
ReStructuredTexy
Projet conçu pour démontrer une structure documentaire
professionnelle basée exclusivement sur ReStructuredText (RST). 
Il sert de
modèle pour tout projet nécessitant une documentation claire, modulaire et
extensible.

Objectifs
---------

- Centraliser toute la documentation dans ``docs/``.
- Fournir une base propre et maintenable.
- Faciliter l’intégration avec Sphinx.
- Offrir un modèle reproductible pour d’autres projets.

Principes
---------

- Simplicité
- Lisibilité
- Modularité
- Extensibilité
