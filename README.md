# 📘 ProjetRST
> Documentation professionnelle en ReStructuredText (RST) pour projets modulaires.  
- ProjetRST sert de modèle pour structurer, organiser et publier une documentation claire,
maintenable et extensible.

---

### 📁 Structure du projet
```text
ProjetRST/
├── docs/
│   ├── index.rst
│   ├── introduction.rst
│   ├── installation.rst
│   ├── usage.rst
│   ├── architecture.rst
│   ├── modules.rst
│   ├── api.rst
│   ├── configuration.rst
│   ├── cli.rst
│   ├── roadmap.rst
│   ├── changelog.rst
│   └── glossary.rst
├── src/
├── tests/
└── README.md
```

---

### 🚀 Objectifs
- Centraliser toute la documentation dans docs/
- Fournir une base propre et professionnelle
- Faciliter la génération Sphinx
- Servir de modèle reproductible pour d’autres projets

---

### 📦 Installation
```bash
pip install projetrst
```

Développement :
```bash
git clone https://github.com/username/projetrst.git
cd projetrst
pip install -e .
```

---

### 🧰 Utilisation
```python
from projetrst import Client

client = Client()
client.run()
```
