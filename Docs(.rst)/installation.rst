================
Installation
================

Prérequis
---------

- Python 3.10 ou supérieur
- Pip ou Poetry
- Environnement virtuel recommandé

Installation via pip
--------------------

.. code-block:: bash

   pip install projetrst

Installation locale (développement)
-----------------------------------

.. code-block:: bash

   git clone https://github.com/username/projetrst.git
   cd projetrst
   pip install -e .
