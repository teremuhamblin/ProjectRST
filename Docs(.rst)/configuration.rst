=================
Configuration
=================

Fichier de configuration
------------------------

ProjetRST utilise un fichier YAML pour la configuration :

.. code-block:: yaml

   app:
     debug: false
     log_level: INFO

Variables d’environnement
-------------------------

- ``PROJetrst_DEBUG`` : active le mode debug
- ``PROJetrst_CONFIG`` : chemin vers un fichier de configuration personnalisé
